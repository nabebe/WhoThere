5. App Description

Who's There? This app allows people to meet up with friends. For instance, if the user is in the same building or close by a friend that you haven’t seen in a certain amount of time, each person would receive a notification informing them that the other friend is in the building/vicinity. This app should be configurable so that it only notifies people when meeting up would be unusual to some degree. For example, family members shouldn’t be notified each time they enter their common home. In addition, this add should be configurable so that the distance at which two people are considered to be in the same place changes.
